# asci-art /output

## this project used to store an ascci art in a file 
## EX: go run main.go --output=<fileName.txt> "hello" standard
## cat fileName.txt
```
 _              _   _          
| |            | | | |         
| |__     ___  | | | |   ___   
|  _ \   / _ \ | | | |  / _ \  
| | | | |  __/ | | | | | (_) | 
|_| |_|  \___| |_| |_|  \___/  
                               

```
## It is impossible to use a different syntax of ascii art.
